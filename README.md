## Comandos básicos de git:

Cambiar de rama:

`git fetch && git checkout branch`

Confirmar un cambio:

`git commit -m "mensaje"`

Enviar cambios al repositorio:

`git push`

Traer cambios del repositorio a local:

`git pull origin master`
